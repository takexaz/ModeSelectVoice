ModeSelectVoise
============================
モードを選択/確定したときに音声を再生

## Requirements
* [Microsoft Visual C++ 再頒布可能パッケージ(x86)](https://aka.ms/vs/17/release/vc_redist.x86.exe)
* [Mebius](https://github.com/takexaz/Mebius/releases/latest)

## Installation
1. [Mebius](https://github.com/takexaz/Mebius/releases/latest)を導入
2. mods/ModeSelectVoiceディレクトリを作成
3.  [ModeSelectVoice.mx](https://github.com/takexaz/ModeSelectVoice/releases/latest/download/ModeSelectVoice.mx), [ModeSelectVoice.toml](https://github.com/takexaz/ModeSelectVoice/releases/download/0.0.1/ModeSelectVoice.toml)ファイルを配置
```
root
└mods
| └Mebius.toml
| └ModeSelectVoice
|   └ModeSelectVoice.mx
|   └ModeSelectVoice.toml
└DINPUT.dll
└Mebius.dll
└winmugen.exe
```
